# أسئلة وأجوبة شائعة - إضافة Dust2 Enhancement 🤔

## 📌 أسئلة عامة

### ❓ إيه هي الإضافة دي؟
**الجواب:** دي إضافة (Plugin) لسيرفرات Counter-Strike 1.6 بتحسن تجربة اللعب على خريطة de_dust2. بتضيف رسائل ترحيب، معلومات على الشاشة، مكافآت، ونصائح تكتيكية.

### ❓ الإضافة بتشتغل على كل الخرائط؟
**الجواب:** لأ، بتشتغل بس على خريطة **de_dust2** فقط. لو غيرت الخريطة، الإضافة بتتعطل تلقائياً.

### ❓ الإضافة مجانية؟
**الجواب:** أيوه، مجانية 100% وممكن تستخدمها على أي سيرفر.

### ❓ هل الإضافة آمنة؟
**الجواب:** أيوه، الكود كله نظيف ومكتوب باحترافية. مفيش أي أكواد ضارة أو backdoors.

## 🔧 أسئلة التثبيت

### ❓ إزاي أحول الملف .sma لـ .amxx؟
**الجواب:** عندك طريقتين:

**الطريقة الأولى (الأسهل):**
1. روح على: https://www.amxmodx.org/websc.php
2. ارفع ملف `dust2_enhancement.sma`
3. اضغط Compile
4. نزل الملف `.amxx`

**الطريقة التانية:**
```bash
cd addons/amxmodx/scripting
./amxxpc dust2_enhancement.sma
```

### ❓ فين أحط الملف المحول؟
**الجواب:** حط ملف `dust2_enhancement.amxx` في المجلد ده:
```
addons/amxmodx/plugins/
```

### ❓ إزاي أفعّل الإضافة؟
**الجواب:** 
1. افتح ملف: `addons/amxmodx/configs/plugins.ini`
2. ضيف السطر ده في آخر الملف:
   ```
   dust2_enhancement.amxx
   ```
3. احفظ الملف
4. أعد تشغيل السيرفر أو غير الخريطة

### ❓ لازم أعيد تشغيل السيرفر؟
**الجواب:** مش لازم تقفل السيرفر كله. ممكن بس تغير الخريطة:
```
amx_map de_dust2
```

## ⚠️ أسئلة حل المشاكل

### ❓ الإضافة مش شغالة، أعمل إيه؟
**الجواب:** جرب الحلول دي بالترتيب:

1. **تأكد إنك على de_dust2:**
   ```
   amx_map de_dust2
   ```

2. **شوف ملف الأخطاء:**
   ```
   addons/amxmodx/logs/error_YYYYMMDD.log
   ```

3. **تأكد إن الإضافة مفعّلة في plugins.ini:**
   - افتح `configs/plugins.ini`
   - تأكد إن السطر موجود: `dust2_enhancement.amxx`
   - متأكد مفيش `;` قبل السطر

4. **تأكد من موديول fun:**
   - افتح `configs/modules.ini`
   - تأكد إن السطر ده موجود ومش معطل:
     ```
     fun_amxx.dll
     ```

### ❓ بيطلع أخطاء في التحويل (Compile Errors)
**الجواب:** 
- استخدم موقع التحويل بدل البرنامج: https://www.amxmodx.org/websc.php
- أو تأكد إن ملفات الـ include موجودة في `scripting/include/`

### ❓ الإضافة بتشتغل على كل الخرائط مش بس Dust2
**الجواب:** ده مش المفروض يحصل! الإضافة فيها كود بيشيك على اسم الخريطة. لو حصل كده:
1. تأكد إنك نزلت الملف الصحيح
2. شوف ملف الـ log
3. ابعت لي تفاصيل المشكلة

### ❓ السيرفر بيقع (Crash) بعد ما حطيت الإضافة
**الجواب:** 
1. شيل الإضافة من plugins.ini مؤقتاً
2. شوف ملف error_*.log
3. تأكد إن إصدار AMX Mod X عندك 1.8.2 أو أحدث:
   ```
   amxx version
   ```

## 🎮 أسئلة الاستخدام

### ❓ إزاي أشوف إن الإضافة شغالة؟
**الجواب:** لما تدخل السيرفر على de_dust2، هتشوف:
- رسالة ترحيب في الشات
- معلوماتك (HP, Armor, Team) على الشاشة شمال
- رسالة في بداية كل راوند
- مكافأة لما تبدأ (+10 HP أو Armor)
- لو حاولت تشتري AWP هيمنعك

### ❓ المكافآت بتيجي إزاي؟
**الجواب:** لما تبدأ (Spawn)، الإضافة بتديك واحدة من اتنين بشكل عشوائي:
- +10 صحة (HP) - الحد الأقصى 110
- +10 درع (Armor) - الحد الأقصى 100

### ❓ ليه AWP ممنوع؟
**الجواب:** عشان اللعب يبقى متوازن أكتر. AWP سلاح قوي جداً على Dust2 (خصوصاً Long A)، ومنعه بيخلي اللعب أكتر حركة وتكتيك.

### ❓ ممكن أشتري Scout بدل AWP؟
**الجواب:** أيوه، Scout مسموح. الممنوع بس AWP.

### ❓ الرسائل بتظهر كل قد إيه؟
**الجواب:** الرسائل (النصائح والرسائل الجوية) بتظهر كل 45-60 ثانية بشكل عشوائي.

### ❓ ممكن أعطل الرسائل؟
**الجواب:** لو عايز تعطل الرسائل، لازم تعدل الكود وتعمل Compile تاني. أو ممكن تعطل الإضافة كلها.

## ⚙️ أسئلة التخصيص

### ❓ عايز أغير قيمة المكافآت، أعمل إيه؟
**الجواب:** 
1. افتح ملف `dust2_enhancement.sma`
2. غير الأرقام دي:
   ```pawn
   #define BONUS_HEALTH_AMOUNT 10    ← مكافأة الصحة
   #define BONUS_HEALTH_MAX    110   ← أقصى صحة
   #define BONUS_ARMOR_AMOUNT  10    ← مكافأة الدرع
   #define BONUS_ARMOR_MAX     100   ← أقصى درع
   ```
3. احفظ الملف
4. اعمل Compile تاني
5. ارفع الملف الجديد على السيرفر

### ❓ عايز أغير مكان المعلومات على الشاشة
**الجواب:**
1. افتح ملف `dust2_enhancement.sma`
2. غير الأرقام دي:
   ```pawn
   #define HUD_STATUS_X  0.02   ← من 0.0 (شمال) لـ 1.0 (يمين)
   #define HUD_STATUS_Y  0.25   ← من 0.0 (فوق) لـ 1.0 (تحت)
   ```
3. اعمل Compile تاني

### ❓ عايز أضيف رسائل جديدة
**الجواب:**
1. افتح ملف `dust2_enhancement.sma`
2. دور على المصفوفات دي:
   ```pawn
   new const g_szTipMessages[][] = { ... }
   new const g_szAtmosphereMessages[][] = { ... }
   ```
3. ضيف رسائلك الجديدة
4. اعمل Compile تاني

### ❓ عايز أسمح بـ AWP
**الجواب:**
1. افتح ملف `dust2_enhancement.sma`
2. دور على الفانكشن `cmd_say`
3. امسح أو علق على الكود اللي بيمنع AWP
4. اعمل Compile تاني
