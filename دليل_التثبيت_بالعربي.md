# دليل تثبيت واستخدام إضافة Dust2 Enhancement - بالعربية 🇪🇬

## 📋 نظرة عامة

الإضافة دي بتحسن تجربة اللعب على خريطة de_dust2 في Counter-Strike 1.6 وبتضيف:
- رسائل ترحيب للاعبين
- عرض معلومات اللاعب على الشاشة (HP, Armor, Team)
- مكافآت عند البداية (+10 صحة أو درع)
- منع سلاح AWP عشان اللعب يبقى متوازن
- رسائل ونصائح تكتيكية خاصة بخريطة Dust2

---

## 🎯 الخطوات المطلوبة (بالترتيب)

### الخطوة 1️⃣: تجهيز الملفات

**المطلوب عندك:**
- سيرفر Counter-Strike 1.6
- AMX Mod X مثبت على السيرفر (إصدار 1.8.2 أو أحدث)
- ملف الإضافة: `dust2_enhancement.sma`

---

### الخطوة 2️⃣: تحويل الملف (Compile)

الملف `.sma` ده ملف كود مصدري، لازم نحوله لملف `.amxx` عشان السيرفر يقدر يشغله.

#### 🔧 طريقة التحويل:

**الطريقة الأولى: باستخدام برنامج التحويل على جهازك**

1. روح على مجلد السكريبتات في السيرفر:
   ```
   addons/amxmodx/scripting
   ```

2. حط ملف `dust2_enhancement.sma` في المجلد ده

3. افتح Terminal أو Command Prompt وروح للمجلد ده:
   ```bash
   cd addons/amxmodx/scripting
   ```

4. شغل أمر التحويل:
   ```bash
   ./amxxpc dust2_enhancement.sma
   ```
   
   أو على Windows:
   ```bash
   amxxpc.exe dust2_enhancement.sma
   ```

5. لو كل حاجة تمام، هيطلع لك ملف اسمه: `dust2_enhancement.amxx`

**الطريقة التانية: باستخدام موقع التحويل (أسهل)**

1. افتح الموقع ده: https://www.amxmodx.org/websc.php

2. اضغط على "Browse" واختار ملف `dust2_enhancement.sma`

3. اضغط "Compile"

4. نزل الملف المحول `dust2_enhancement.amxx`

---

### الخطوة 3️⃣: تثبيت الإضافة على السيرفر

1. **انقل الملف المحول:**
   - خد ملف `dust2_enhancement.amxx` اللي اتعمل
   - حطه في المجلد ده:
     ```
     addons/amxmodx/plugins/
     ```

2. **فعّل الإضافة:**
   - افتح الملف ده:
     ```
     addons/amxmodx/configs/plugins.ini
     ```
   
   - ضيف السطر ده في آخر الملف:
     ```
     dust2_enhancement.amxx
     ```
   
   - احفظ الملف

---

### الخطوة 4️⃣: إعادة تشغيل السيرفر

**اختار واحدة من الطرق دي:**

**الطريقة الأولى: إعادة تشغيل السيرفر كله**
- أقفل السيرفر وشغله تاني

**الطريقة التانية: تغيير الخريطة**
- افتح الكونسول واكتب:
  ```
  amx_map de_dust2
  ```

## ✅ التأكد إن الإضافة شغالة

بعد ما تخلص الخطوات، ادخل على السيرفر وتأكد من:

### 1. رسالة الترحيب
لما تدخل السيرفر، لازم تشوف رسالة في الشات زي:
```
[DUST2 ENHANCED] Welcome to DUST2 ENHANCED, [اسمك]!
```

### 2. معلومات اللاعب على الشاشة
على الجانب الشمال من الشاشة، هتلاقي معلوماتك:
```
HP: 100
AR: 100
Team: CT
```
- اللون أخضر لو صحتك فوق 75
- اللون أصفر لو صحتك بين 40-75
- اللون أحمر لو صحتك تحت 40

### 3. رسالة بداية الراوند
في بداية كل راوند، هتظهر رسالة في أعلى الشاشة زي:
```
DUST2 ENHANCED * Round 1 * Secure Long A!
```

### 4. مكافأة البداية (Spawn Bonus)
لما تبدأ، هتاخد واحدة من دول:
- `+10 HP * Desert Blessing` (صحة إضافية)
- `+10 Armor * Fortified` (درع إضافي)

### 5. منع AWP
لو حاولت تشتري AWP، هتظهر رسالة:
```
AWP restricted - Master the AK/M4 instead!
```

### 6. النصائح والرسائل
كل 45-60 ثانية، هتظهر رسائل في الشات زي:
- نصائح تكتيكية: `[TIP] Smoke Xbox to safely cross Mid to B`
- رسائل جوية: `[DUST2] The desert sun beats down on ancient stones`

## 🔧 حل المشاكل الشائعة

### المشكلة: الإضافة مش شغالة

**الحلول:**

1. **تأكد إنك على خريطة de_dust2**
   - الإضافة بتشتغل بس على Dust2
   - جرب تغير الخريطة: `amx_map de_dust2`

2. **شوف ملف الأخطاء**
   - افتح: `addons/amxmodx/logs/error_YYYYMMDD.log`
   - لو فيه أخطاء، هتلاقيها مكتوبة هنا

3. **تأكد من إصدار AMX Mod X**
   - اكتب في الكونسول: `amxx version`
   - لازم يكون 1.8.2 أو أحدث

4. **تأكد إن الموديول `fun` مفعّل**
   - افتح: `addons/amxmodx/configs/modules.ini`
   - تأكد إن السطر ده موجود ومش عليه `;`:
     ```
     fun_amxx.dll
     ```
     أو على Linux:
     ```
     fun_amxx_i386.so
     ```

### المشكلة: فيه أخطاء في التحويل (Compile Errors)

**الحلول:**

1. تأكد إن ملفات الـ include موجودة:
   - `amxmodx.inc`
   - `amxmisc.inc`
   - `fun.inc`
   
   لازم تكون في: `addons/amxmodx/scripting/include/`

2. استخدم موقع التحويل بدل البرنامج:
   - https://www.amxmodx.org/websc.php

### المشكلة: الإضافة شغالة على كل الخرائط

**الحل:**
- ده مش المفروض يحصل! الإضافة بتشيك على اسم الخريطة
- لو حصل، ابعت لي ملف الـ log عشان أشوف المشكلة

## 🎨 تخصيص الإضافة (اختياري)

لو عايز تغير حاجة في الإضافة، افتح ملف `dust2_enhancement.sma` وغير الأرقام دي:

### تغيير قيمة المكافآت:
```pawn
#define BONUS_HEALTH_AMOUNT 10    ← غير الرقم ده (مكافأة الصحة)
#define BONUS_HEALTH_MAX    110   ← غير الرقم ده (أقصى صحة)
#define BONUS_ARMOR_AMOUNT  10    ← غير الرقم ده (مكافأة الدرع)
#define BONUS_ARMOR_MAX     100   ← غير الرقم ده (أقصى درع)
```

### تغيير مكان المعلومات على الشاشة:
```pawn
#define HUD_STATUS_X  0.02   ← من 0.0 (شمال) لـ 1.0 (يمين)
#define HUD_STATUS_Y  0.25   ← من 0.0 (فوق) لـ 1.0 (تحت)
```

**ملحوظة:** بعد أي تعديل، لازم تعمل Compile تاني للملف!
