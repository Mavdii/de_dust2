📌 الخطوات بالترتيب (خطوة بخطوة)
═══════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────┐
│ الخطوة 1: تحويل الملف (Compile)                                        │
└─────────────────────────────────────────────────────────────────────────┘

🔹 الطريقة السهلة (باستخدام الموقع):

   1. افتح المتصفح وروح على:
      https://www.amxmodx.org/websc.php

   2. هتشوف صفحة فيها:
      ┌─────────────────────────────┐
      │  AMX Mod X Web Compiler     │
      │                             │
      │  [Browse...]  [Compile]     │
      └─────────────────────────────┘

   3. اضغط على "Browse" واختار ملف:
      dust2_enhancement.sma

   4. اضغط "Compile"

   5. انتظر شوية (5-10 ثواني)

   6. هيظهر لك رابط تحميل:
      ┌─────────────────────────────┐
      │ ✅ Compilation successful!  │
      │                             │
      │ [Download .amxx file]       │
      └─────────────────────────────┘

   7. اضغط على الرابط ونزل الملف
      اسمه: dust2_enhancement.amxx

═══════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────┐
│ الخطوة 2: رفع الملف على السيرفر                                        │
└─────────────────────────────────────────────────────────────────────────┘

🔹 باستخدام FTP أو File Manager:

   1. افتح برنامج FTP (مثل FileZilla) أو لوحة التحكم

   2. روح على مجلد السيرفر:
      cstrike/addons/amxmodx/plugins/

   3. ارفع ملف dust2_enhancement.amxx في المجلد ده

   4. تأكد إن الملف اترفع صح:
      ┌─────────────────────────────────────┐
      │ 📁 plugins/                         │
      │   📄 admin.amxx                     │
      │   📄 admincmd.amxx                  │
      │   📄 dust2_enhancement.amxx  ← ده   │
      │   📄 ...                            │
      └─────────────────────────────────────┘

═══════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────┐
│ الخطوة 3: تفعيل الإضافة                                                │
└─────────────────────────────────────────────────────────────────────────┘

🔹 تعديل ملف plugins.ini:

   1. روح على المجلد:
      cstrike/addons/amxmodx/configs/

   2. افتح ملف: plugins.ini

   3. هتلاقي محتوى الملف كده:
      ┌─────────────────────────────────────┐
      │ ; AMX Mod X plugins                 │
      │                                     │
      │ admin.amxx                          │
      │ admincmd.amxx                       │
      │ adminhelp.amxx                      │
      │ ...                                 │
      └─────────────────────────────────────┘

   4. روح لآخر الملف وضيف السطر ده:
      dust2_enhancement.amxx

   5. الملف بعد التعديل:
      ┌─────────────────────────────────────┐
      │ ; AMX Mod X plugins                 │
      │                                     │
      │ admin.amxx                          │
      │ admincmd.amxx                       │
      │ adminhelp.amxx                      │
      │ ...                                 │
      │ dust2_enhancement.amxx       ← ده   │
      └─────────────────────────────────────┘

   6. احفظ الملف (Ctrl+S أو Save)

═══════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────┐
│ الخطوة 4: إعادة تشغيل السيرفر                                          │
└─────────────────────────────────────────────────────────────────────────┘

🔹 اختار طريقة:

   الطريقة الأولى (الأسهل):
   ─────────────────────────
   1. افتح الكونسول في اللعبة
   2. اكتب: amx_map de_dust2
   3. اضغط Enter

   الطريقة التانية:
   ─────────────────────────
   1. أقفل السيرفر
   2. شغله تاني
   3. غير الخريطة لـ de_dust2

═══════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────┐
│ الخطوة 5: التأكد إن الإضافة شغالة                                      │
└─────────────────────────────────────────────────────────────────────────┘

   ✅ رسالة الترحيب في الشات:

      └─────────────────────────────────────────────┘

   ✅ مكافأة البداية (وسط الشاشة):

❌ مشاكل شائعة وحلولها
═══════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────┐
│ المشكلة: الإضافة مش شغالة                                              │
└─────────────────────────────────────────────────────────────────────────┘

   الحل 1: تأكد إنك على خريطة de_dust2
   ────────────────────────────────────────
   الإضافة بتشتغل بس على Dust2!
   غير الخريطة: amx_map de_dust2

   الحل 2: شوف ملف الأخطاء
   ────────────────────────────────────────
   افتح: addons/amxmodx/logs/error_YYYYMMDD.log
   لو فيه أخطاء هتلاقيها مكتوبة

   الحل 3: تأكد من الموديول fun
   ────────────────────────────────────────
   افتح: addons/amxmodx/configs/modules.ini
   تأكد إن السطر ده موجود:
   fun_amxx.dll (Windows)
   أو
   fun_amxx_i386.so (Linux)

┌─────────────────────────────────────────────────────────────────────────┐
│ المشكلة: فيه أخطاء في التحويل                                          │
└─────────────────────────────────────────────────────────────────────────┘

   الحل: استخدم موقع التحويل
   ────────────────────────────────────────
   https://www.amxmodx.org/websc.php
   أسهل وأضمن!

💡 نصائح مهمة
═══════════════════════════════════════════════════════════════════════════

   1. احتفظ بنسخة من الملف الأصلي (.sma) عشان لو عايز تعدل حاجة

   2. لو عايز تعطل الإضافة مؤقتاً، حط ; قبل السطر في plugins.ini:
      ;dust2_enhancement.amxx

   3. الإضافة آمنة 100% ومش هتأثر على اللعب

   4. لو عندك مشكلة، شوف ملف error_*.log الأول

   5. الإضافة بتشتغل تلقائي، مش محتاج أي إعدادات إضافية
