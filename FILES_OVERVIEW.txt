🚀 QUICK START
═══════════════════════════════════════════════════════════════════════════

1. COMPILE
   ./compile.sh
   (or: amxxpc dust2_enhancement.sma)

2. INSTALL
   cp dust2_enhancement.amxx addons/amxmodx/plugins/

3. ENABLE
   Add to plugins.ini: dust2_enhancement.amxx

4. RESTART
   Restart server or: amx_map de_dust2
